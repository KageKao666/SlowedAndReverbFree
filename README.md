# 💕 SlowedAndReverbFree

> Поддержать проект: https://yoomoney.ru/fundraise/1CV7BFDQ964.250923

Полностью бесплатный аудио-редактор для создания slowed & reverb версий треков прямо в браузере.

## Доступные эффекты:

- **Скорость** 🎵
- **Высота тона** 🎶
- **Реверберация** 🌊
- **Bass boost** 🔊
- **Under water** 🐠
